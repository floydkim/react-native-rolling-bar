# react-native-rolling-bar

Rolling bar (banner) UI for React Native

## Installation

```sh
npm install react-native-rolling-bar
```

## Usage

```js
import { multiply } from "react-native-rolling-bar";

// ...

const result = await multiply(3, 7);
```
